▸ UIMP — Universal Integration Migration Platform

Developed an any-to-any iPaaS migration platform at Accenture Research and Development that automates the migration of integration workflows across 150+ heterogeneous integration platforms, using AI-driven agents and event-driven orchestration.

  • Led the Data Plane end-to-end — owned design, implementation, and delivery of the entire write-path event publishing infrastructure powering all UIMP agents.

  • Defined 9 canonical event types with strict schema validation, eliminating an entire class of production bugs from typos, missing fields, and prohibited payload data.
  • Built RabbitMQ publisher infrastructure with quorum queues, async pipelined confirms, lock-free channel pools, and graceful 3-state shutdown — sustaining 10,000+ msg/sec per instance.
